### Manual Command

- *NOTICE*: ``[  ]`` means optional, while ``< >`` means mandatory
- ``man -k which`` - "k" option deals with searching
- ``man which`` - opens  **User Commands** (short cut)
- ``which [-a] filename ...`` 
- ``mkdir``  - make a new directory
- ``touch`` - make a new file
- ``rm`` - remove file
- ``rm -rf`` - remove folder
